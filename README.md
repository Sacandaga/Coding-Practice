# National Security Simulation — Project Portfolio

Academic NSC principals-level simulation covering U.S. decision-making during the 2025–2026 Houthi Red Sea crisis and the 2026 Iran War.

## Deliverables

### Interactive tools
| Page | Description |
|------|-------------|
| [Strategic analysis dashboard](dashboard.html) | 6 predictions, 4-slider scenario model, probability forecasts, cost analysis |
| [Weapons systems explorer](weapons-explorer.html) | 12 platforms, clickable specs, range rings on theater map, strike package calculator |
| [Theater map](theater-map.html) | Pan/zoom map, 11 toggleable layers, 18 clickable markers, animated vectors |
| [Escalation simulator](escalation-simulator.html) | 10-rung ladder, branching decisions, live dashboard tracking 5 metrics |
